A Playlist for Anything

Overview

Duration: one week, July 2023

Solo project

Tools: ReactJS, Node.js, Express, Socket.IO, Spotify API

This is a multiplayer game that presents users with a randomized prompt and challenges them to create a playlist following the prompt under a self-selected time. Once time is up, they are able to see each others playlists.

Why?

My friends and I love making random playlists and playlists for each other. I thought it would be fun to be able to gameify it on a web app!

Challenges

This was my first time working with sockets and back-end on web-apps, so there was a lot of unknown but it was incredibly fun learning more through this project!

Future

At this time, it supports 2 players. But I would love to allow up to 4 players in the future!
